[Wiki] NovaLib | Overview

NovaLib is an API and library for Bukkit and Bungee plugins to build upon. Designed to be simple to use and install without sacrificing any functionality or features.
NovaLib requires no additional dependencies or plugins in order to function and is completely self-contained.
