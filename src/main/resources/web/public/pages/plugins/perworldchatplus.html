PerWorldChatPlus

Run a server with multiple worlds? Hate having the chat go to all worlds?
Well, PerWorldChatPlus is the right plugin for you.

Not only can this plugin make chats perworld, it allows for grouping, global chat spying on all chats, and more!
Perfect for all servers who don't want chats to be in all worlds. Large or small.
